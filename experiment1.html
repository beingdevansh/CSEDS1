Course Curriculum

B.Tech. in Data Science

This is the course curriculum for Bachelor of Technology in Data Structures and Algorithms with specialization in Data Science. The program is designed for 1st year & includes theory as well as practical courses.

  1. SEMESTER I: Foundational Concepts

    • Introduction to C++ Programming
      Course Code: 25CS101
      Credits: 4
      Key Topics:
      • Programming basics C++
      • Arrays
      • Introduction to Object Oriented Programming
    • Linear Algebra & Calculus
      Course Code: 25AS102
      Credits:4
      Key Topics:
      • Vectors and Matrices
      • Derivatives and Integrals
      • Eigenvalues and Eigenvectors

  2. SEMESTER II: Object Oriented Programing

    • Data Structures & Algorithms
      Course Code: CS102
      Credits: 4
      Key Topics:
      • Arrays, Stacks, Queues, and Linked Lists
      • Trees and Graphs
      • Sorting and Searching Algorithms
    • Foundations of Machine Learning
      Course Code: 25AI102
      Credits: 3
      Key Topics:
      • Supervised vs. Unsupervised Learning
      • Linear & Logistic Regression
      • Introduction to Neural Networks
